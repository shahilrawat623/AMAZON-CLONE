# Amazon Clone 🛒

A simple **Amazon homepage clone** built with **HTML and CSS only**.  
This project is designed to practice **layout building, and component styling** without JavaScript.

---

## 🚀 Features
-  **navbar** with logo, search bar, and navigation links
- **Hero banner** section with promotional content
- **Product grid** layout using CSS Grid/Flexbox
- Clean, modern **footer**
- Design for desktop only

---

## 🛠️ Tech Stack
- **HTML5** for structure
- **CSS3** for styling

---

## 📂 Project Structure
amazon-clone/<br>
│── index.html<br>
│── style.css<br>
│── assets/ (images, icons, logos)

---

## 🎯 Learning Goals
- Practice **semantic HTML** structure
- Build layouts with **Flexbox** and **CSS Grid**
- Improve front‑end project organization
